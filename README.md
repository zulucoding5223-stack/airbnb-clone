# airbnb-clone
A full-stack Airbnb clone built as a final project using the MERN stack, featuring authentication, listings, reservations, and an admin dashboard.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- CSS / Tailwind

## Features
- User authentication (login & register)
- Property listings
- Booking functionality
- Host dashboard
